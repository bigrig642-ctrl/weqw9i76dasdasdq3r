Getting Started

Install, configure, and publish

These steps cover both the Discord bot setup flow and the static GitHub Pages docs added to this repository.

Prerequisites

  • Node.js 18.18 or newer
  • A Discord bot token and application client ID
  • Channel and role IDs for the setup systems you want to use
  • GitHub Pages enabled if you want to publish the docs site

Install

Install bot dependencies from the repository root.

npm install
Tip: For hosting panels such as Bisect Hosting, upload the repository and run the install there instead of uploading a local node_modules folder.

Configure

Copy the example environment file and fill in the required IDs.

copy .env.example .env
TOKEN=your_bot_token
CLIENT_ID=your_application_id
GUILD_ID=your_test_guild_id
AUTO_REGISTER_COMMANDS=true

Once the bot is online, use the setup commands in Discord to finish channel, role, ticket, application, giveaway, and vouch configuration.

Register Commands

Register slash commands manually if you are not using automatic registration on boot.

npm run register
npm start

After the bot starts, configure the main systems with: /setuproles, /setupchannels, /setuptickets, /setupapps, /setupwelcome, and /setupvouch.

Publish Docs

The docs site is fully static. Commit the HTML, CSS, and JavaScript files to your repository and enable GitHub Pages.
